BETCHAIN is a sports prediction platform where users can deposit ETH into liquidity pools, place bets on sporting events, and receive payouts based on oracle-provided results. The platform combines blockchain smart contracts with a modern front-end for real-time interactions.

---

## Features

- **Liquidity Pool Management:** Users can securely deposit and withdraw ETH.
- **Prediction Market:** Place bets on sports events with multiple possible outcomes.
- **Oracle Integration:** Event results provided by a Sports Oracle smart contract.
- **Payout System:** Winners automatically receive payouts from the liquidity pool.
- **Front-End Interface:** Built with React + Vite, supporting wallet connection and real-time updates.
- **Event Management:** Resolve matches and execute payouts using smart contracts.
